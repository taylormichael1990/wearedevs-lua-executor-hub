## Overview

WeAreDevs is a Windows-only Roblox script executor built for users who want a straightforward desktop tool for Lua execution and script hub browsing. The app centers on a compact WPF interface, fast startup, and a workflow that keeps script launching simple and free of extra setup.

It also ships with auto-update support, a persistent script queue, and SQLite-backed local storage for saved sessions and script management. With Windows 10/11 support and a no-key approach, it is presented as a lightweight Roblox executor with integrated script tooling.
